# 某 Jun 所智能问答

> 标签：QA · 内网部署
> 类别：AI 智能应用
> 序号：11

## 项目简介

完全私有化部署的大模型问答系统，适配涉密内网环境。将领域业务知识（手册、规程、历史案例）通过 RAG 方式进行增强，支持自然语言提问并生成可溯源的高质量答案，满足科研/工程人员在安全环境下的知识获取需求。
