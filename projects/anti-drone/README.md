# 反无人机监控系统

> 标签：安防 · CV
> 类别：AI 智能应用
> 序号：07

## 项目简介

融合可见光、红外与雷达等多模态感知数据，实时识别低空小型无人机目标。结合轨迹预测与行为分析算法，对非法入侵、黑飞等行为提前预警并联动处置，显著提升重要区域低空安防的自动化与智能化水平。
