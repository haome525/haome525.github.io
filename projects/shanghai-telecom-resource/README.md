# 上海电信网络资源管理平台

> 标签：电信 · 资源管理
> 类别：企业信息化
> 序号：16

## 项目简介

面向电信运营商的全生命周期网络资源管理系统，涵盖光缆、管道、机房设备、端口及链路等物理与逻辑资源。采用统一资源模型与自动拓扑发现技术，实现资源关联关系可视化；支持资源调度、故障定位与影响分析，结合GIS地图展示资源地理分布。系统日均处理上万条资源变更，有效提升资源利用率与运维效率，支撑上海地区百万级用户网络服务。
