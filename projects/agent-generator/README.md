# 某智能体生成器系统

> 标签：Agent · MCP
> 类别：AI 智能应用
> 序号：13

## 项目简介

基于多智能体协作框架（MCP）的通用 Agent 生成平台。支持动态拆分任务、调度多个 SKILLS 模块协同执行，具备任务规划、工具调用、记忆管理与推理决策能力。用户可通过自然语言快速生成专用智能体，降低开发门槛。
