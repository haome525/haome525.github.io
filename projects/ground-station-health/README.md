# 某地面站健康管理系统

> 标签：航天 · 地面站
> 类别：航天卫星

## 项目简介

面向测控站点设备体系的健康管理方案。基于知识图谱建模设备关系，实时监视天伺馈信记基带等多类设备健康状态；提供设备级健康预测、故障智能诊断与根因定位，自动推送维修建议。打通全要素数据，实现地面站可视化智能运维。
