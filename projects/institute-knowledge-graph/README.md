# 某院所知识图谱系统

> 标签：图谱 · NLP
> 类别：AI 智能应用
> 序号：09

## 项目简介

基于自然语言处理与命名实体识别技术，从海量非结构化文本中自动抽取实体、属性及关系，构建领域知识图谱。提供图数据库存储与可视化探索工具，支持多跳推理、关联分析与知识溯源，赋能军工/科研场景的智能情报挖掘。

## 说明

本文件夹为「某院所知识图谱系统」子项目独立存储单元。

- `README.md`：项目说明（Markdown 源，本文件）
- `index.html`：项目详情页（modern-page 布局，复用全局主题系统），访问路径 `/projects/institute-knowledge-graph/`
