# 某卫星测运控项目

> 标签：航天 · 测运控系统
> 类别：航天卫星
> 序号：02

## 项目简介

多型号卫星测运控一体化平台，覆盖遥测解析、遥控指令编排、轨道预报与计算、姿态控制及设备资源调度。打通天地链路与地面应用闭环，支持多任务并行调度与关键参数实时监视；融入 AI 模型辅助异常检测与轨道预报，控制周期亚秒级响应，已在多颗在轨卫星中稳定运行。
