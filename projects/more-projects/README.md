# 更多其他项目

> 标签：综合 · 信息化
> 类别：企业信息化
> 序号：18

## 项目简介

参与多个行业信息化建设项目，涵盖政务大数据共享交换、智能运维监控（AIOps）、智慧园区物联网平台等。技术栈包括微服务架构（Spring Cloud）、实时流计算（Flink）、时序数据库（InfluxDB）及可视化大屏，实现数据采集、治理、分析与决策支持一体化。项目均顺利交付并稳定运行，助力客户实现数字化转型与精细化管理。
