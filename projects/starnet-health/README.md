# 星网健康管理系统

> 标签：航天 · 卫星健康管理
> 类别：航天卫星
> 序号：03

## 项目简介

卫星星座全生命周期健康管理平台，融合规则引擎、机器学习与大模型推理。涵盖状态监视、趋势预测、故障诊断定位、知识图谱、多级告警、遥控数据订阅分发及处置策略建议等模块，实现从数据到决策的智能升级，支撑批量卫星少人值守，降低运维成本。
