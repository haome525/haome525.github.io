# 某文档自动生成平台

> 标签：AIGC · LLM  Agent
> 类别：AI 智能应用
> 序号：12

## 项目简介

结合结构化模板与大语言模型（LLM）的智能文档生产线。支持多格式输入（Word、PDF、Excel）与自定义模板配置，由 LLM Agent 自动填充变量、润色段落并生成符合规范的军标/企标文档，大幅提高写作效率与一致性。
