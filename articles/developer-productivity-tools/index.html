程序员的效率工具清单

haome525 2026-06-16 6分钟阅读

IDE与编辑器

VS Code是目前最流行的代码编辑器，拥有丰富的插件生态。JetBrains系列IDE（IntelliJ IDEA、WebStorm、PyCharm）在特定语言领域表现出色，提供强大的代码分析和重构能力。

终端工具

现代终端工具如Warp、Tabby、Windows Terminal等提供了更好的用户体验。oh-my-zsh、tmux、Starship等工具可以大幅提升终端使用效率。

效率工具

Raycast（Mac）、uTools等启动器可以快速启动应用和执行操作。Notion、Obsidian、Logseq等笔记工具帮助管理知识和笔记。

开发辅助

GitHub Copilot、Cursor、Codeium等AI编程助手正在改变开发方式。Postman、Insomnia等API工具简化了接口调试。Docker Desktop简化了容器管理。

协作工具

Git是版本控制的基础，GitHub/GitLab提供代码协作平台。Linear、Jira等项目管理工具帮助团队协作。Figma是设计协作的首选工具。
