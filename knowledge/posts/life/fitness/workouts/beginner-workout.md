# 新手健身入门计划

## 1. 核心原则

- **渐进超负荷**：每周增加重量/次数/组数
- **训练频率**：每个肌群每周训练 2 次
- **恢复**：睡眠 7-8 小时，蛋白质摄入充足

## 2. 训练计划（推拉腿 PPL）

### 推日（胸、肩、三头）

```
杠铃卧推        4×8-12
哑铃推举        3×10-15
坐姿夹胸        3×12-15
绳索下压        3×12-15
```

### 拉日（背、二头）

```
引体向上        3×力竭
杠铃划船        4×8-12
坐姿划船        3×12-15
哑铃弯举        3×10-12
```

### 腿日

```
杠铃深蹲        4×8-12
罗马尼亚硬拉    3×10-12
腿举            3×12-15
小腿提踵        4×15-20
```

## 3. 营养建议

- 蛋白质：每公斤体重 1.6-2.0g
- 碳水：训练前后补充
- 热量：增肌 +200~300 大卡，减脂 -300~500 大卡
