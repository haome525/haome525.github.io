# 程序员理财投资入门

## 1. 财务健康金字塔

```
        财务自由
        ╱      ╲
      投资增值  被动收入
      ╱          ╲
    应急基金      保险保障
        ╲        ╱
      收入来源 + 预算管理
```

## 2. 资产配置（参考）

| 资产类别 | 比例 | 说明 |
|----------|------|------|
| 货币基金 | 10% | 3-6 个月生活费，随时可取 |
| 指数基金 | 50% | 沪深 300 + 中证 500 + 标普 500 |
| 债券基金 | 20% | 降低波动，稳定收益 |
| 行业基金/个股 | 15% | 科技、AI 等自己熟悉的领域 |
| 现金 | 5% | 抓住机会的弹药 |

## 3. 指数基金定投

```python
# 每月定投策略
每月收入 = 30000
定投比例 = 0.3
定投金额 = 每月收入 * 定投比例

# 分散到不同指数
沪深300    = 定投金额 * 0.4  # 大盘蓝筹
中证500    = 定投金额 * 0.3  # 中盘成长
标普500    = 定投金额 * 0.3  # 美股
```

## 4. 避坑指南

- ❌ 不要追涨杀跌
- ❌ 不要 All in 单一资产
- ❌ 不要碰不懂的投资品（虚拟货币土狗、P2P）
- ✅ 定投 + 长期持有
- ✅ 闲钱投资，不加杠杆
- ✅ 持续学习，建立自己的投资体系

## 5. 程序员优势

- 收入较高，现金流充裕
- 理解技术驱动行业，AI/科技股有认知优势
- 逻辑思维强，适合量化投资
- 可以开发个人理财工具/自动化定投脚本
